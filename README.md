# tntjs

#### 介绍
tnt.js是一个基于javascript的编程语言 目标是兼容javascript的功能 将语法系统化 降低学习成本

#### 样例

```html
<html>
<head>
<script scr="tnt.min.js"></script>
</head>
<body>
<tnt>
print("<h5>Hello world!</h5>");
</tnt>
</div></body>
</html>
```
